- Teknik pewarnaan dasar (Bleaching, Toning, Coloring)

- Perawatan dan kesehatan rambut (Hair Mask, Spa, Creambath)



**Fasilitas :**


- Modul/Buku panduan lengkap Tata Kecantikan Rambut Level Dasar & Terampil
